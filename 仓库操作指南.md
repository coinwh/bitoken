# Bitoken 仓库操作指南

## 仓库信息

- **仓库地址**: https://github.com/coinwh/bitoken/
- **分支**: main
- **协议**: 开源项目

## 下载仓库

### 方法一：Git 克隆（推荐）

```bash
git clone https://github.com/coinwh/bitoken.git
```

### 方法二：下载 ZIP

访问仓库地址，点击绿色的 **Code** 按钮，选择 **Download ZIP** 即可下载整个仓库。

## 更新本地仓库

### 拉取最新代码

```bash
git pull origin main
```

## 推送更改

### 基本推送流程

```bash
# 1. 查看当前状态
git status

# 2. 添加所有更改
git add .

# 3. 提交更改
git commit -m "你的提交说明"

# 4. 推送到远程仓库
git push origin main
```

## 常用 Git 命令

| 命令 | 说明 |
|------|------|
| `git status` | 查看当前工作区状态 |
| `git log --oneline -10` | 查看最近 10 条提交记录 |
| `git diff` | 查看未暂存的更改 |
| `git branch` | 查看当前分支 |
| `git stash` | 暂存本地修改 |
| `git stash pop` | 恢复暂存的修改 |

## 常见问题

### 推送被拒绝？

可能是远程代码有更新，先执行 `git pull` 再推送：

```bash
git pull origin main
git push origin main
```

### 本地修改被覆盖？

使用 stash 暂存后再拉取：

```bash
git stash
git pull origin main
git stash pop
```
