# Bitoken

> 毕业设计项目 - 致力于打造一个类似 Cherry Studio 的 AI 客户端应用

**仓库地址**: https://github.com/coinwh/bitoken/

## 项目简介

本项目是我的毕业设计，目标是创建一个功能完善的 AI 桌面客户端，参考 [Cherry Studio](https://github.com/cherryhq/cherry-studio) 的设计理念与功能特性。
