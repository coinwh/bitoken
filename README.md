# Bitoken

> 致力于打造一个类似 Cherry Studio 的 AI 客户端应用

**仓库地址**: https://github.com/coinwh/bitoken/

## 项目简介

本项目目标是创建一个功能完善的 AI 桌面客户端，参考 [Cherry Studio](https://github.com/cherryhq/cherry-studio) 的设计理念与功能特性。

## 寻找开发者

目前项目正在寻找志同道合的开发者一起参与开发。如果你对 AI 客户端应用感兴趣，欢迎加入一起完善这个项目！

## 如何参与

1. Fork 本仓库
2. 创建你的功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 提交 Pull Request
