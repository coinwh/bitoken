## 项目简介

本项目目标是创建一个功能完善的 AI 桌面客户端，参考 [Cherry Studio](https://github.com/cherryhq/cherry-studio) 的设计理念与功能特性。

### 什么是 Cherry Studio？

Cherry Studio 是一款开源的 AI 桌面客户端应用，支持多种大语言模型，提供简洁直观的交互界面。主要特点包括：

- **多模型支持**: 支持 OpenAI、Claude、Gemini 等多种主流 AI 模型
- **本地部署**: 数据存储在本地，保护隐私安全
- **跨平台**: 支持 Windows、macOS、Linux 等主流操作系统
- **开源免费**: 基于 MIT 协议开源，可自由使用和修改
- **丰富功能**: 支持对话管理、知识库、翻译助手等多种实用功能

Cherry Studio 以其优秀的设计理念和强大的功能，成为了 AI 客户端领域的标杆产品之一。

## 技术栈

本项目基于现代前端技术栈构建：

- **Node.js**: 提供后端运行环境
- **React**: 用于构建用户界面的 JavaScript 库
- **Vite**: 新一代前端构建工具，快速的开发服务器和打包体验

通过这些技术，项目实现了高效的开发流程和良好的用户体验。
